# Project Name & Description

## Project Name
**cleansys**

## Short Description
A minimalist CLI tool for systematic digital decluttering through criteria-based file scanning and interactive cleanup.

## Tagline
"Clean your system, one decision at a time."

## GitHub Description (160 chars)
Minimalist CLI tool to identify and remove digital clutter. Scan by age/size, review interactively, clean systematically. Safe, simple, focused.

## PyPI Description
cleansys is a minimalist command-line tool that helps you systematically clean up digital clutter. Scan folders by criteria (age, size, type), review files interactively, and execute cleanup operations with confidence. Designed for simplicity, safety, and focus.

## Elevator Pitch
cleansys helps you reclaim disk space and mental clarity by making digital decluttering systematic rather than overwhelming. Point it at any folder, set your criteria, and review files one by one with full context before deciding what stays and what goes.

## Key Value Propositions
1. **Systematic over random**: Criteria-based scanning eliminates guesswork
2. **Safe by default**: Every action requires explicit approval
3. **Focused workflow**: One decision at a time, no overwhelm
4. **Zero configuration**: Works immediately, no setup needed
5. **Transparent operations**: See exactly what happens, when
